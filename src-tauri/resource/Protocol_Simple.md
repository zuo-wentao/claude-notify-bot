# 阀门推杆通信协议

## 基本信息

- **类型**：文本协议
- **波特率**：115200 bps，8N1
- **结束符**：换行符 `\n` 或回车符 `\r`

---

## 命令列表

| 命令 | 功能 | 示例 |
|------|------|------|
| `START push release count interval duty` | 启动推杆 | `START 50 30 2 100 50` |
| `STOP` | 停止推杆 | `STOP` |
| `LED mode r g b speed` | 设置LED | `LED 2 255 85 0 2000` |
| `QUERY` | 查询状态 | `QUERY` |

---

## START 命令

**格式**：`START push release count interval duty`

| 参数 | 范围 | 说明 |
|------|------|------|
| push | 10~10000 ms | 推进时长 |
| release | 10~10000 ms | 释放时长 |
| count | 1~10000 | 往复次数 |
| interval | 0~10000 ms | 间隔（0=无） |
| duty | 1~100 % | PWM占空比 |

**示例**：`START 500 300 10 100 80`
含义：推进500ms → 释放300ms → 等待100ms → 循环10次，占空比80%

---

## STOP 命令

**格式**：`STOP`

立即停止推杆，输出低电平。

---

## LED 命令

**格式**：`LED mode r g b speed`

| 参数 | 范围 | 说明 |
|------|------|------|
| mode | 0~2 | 0=关闭, 1=常亮, 2=呼吸灯 |
| r/g/b | 0~255 | RGB颜色分量 |
| speed | 100~60000 ms | 呼吸灯周期（越小越快） |

**示例**：
```
LED 0 0 0 0 0      → LED关闭
LED 1 255 0 0 0    → 红色常亮
LED 2 0 255 0 2000 → 绿色呼吸，周期2秒
```

---

## QUERY 命令

**格式**：`QUERY`

**返回**：`STS:running,count,state`

| 字段 | 说明 |
|------|------|
| running | 0=空闲, 1=运行中 |
| count | 当前已完成往复次数 |
| state | 0=IDLE, 1=PUSHING, 2=RELEASING, 3=WAITING |

**示例**：`STS:1,3,1` → 运行中，已完成3次，正在推进

---

## 典型命令

```python
# 启动推杆（快速脉冲）
"START 50 30 2 100 50"

# 启动推杆（深度往复）
"START 1000 800 5 500 100"

# 停止推杆
"STOP"

# LED设置
"LED 0 0 0 0 0"      # 关闭
"LED 1 255 0 0 0"    # 常亮红色
"LED 2 255 85 0 2000"  # 橙黄色呼吸，周期2秒

# 查询状态
"QUERY"
```
